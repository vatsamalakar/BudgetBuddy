# Budget Buddy

Budget Buddy is a simple Flask web app for tracking income, expenses, and spending by category.

## Features

- Register and log in
- Add income and expense transactions
- View total income, expenses, and balance
- See transaction history
- View spending reports with charts
- Deploys to Vercel without extra database setup for demo use

## Try it out

```text
https://budget-buddy-alpha-three.vercel.app/
```

## Tech Stack

- Python
- Flask
- Flask-Login
- Flask-SQLAlchemy
- Bootstrap
- Chart.js
- Vercel
